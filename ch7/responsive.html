Setting the Viewport

The viewport meta tag ensures proper scaling on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

width=device-width - Sets the width of the viewport to the device's screen width

initial-scale=1.0 - Sets the initial zoom level when the page is first loaded

Media Queries

CSS media queries allow you to apply different styles based on device characteristics.

/* Mobile styles */ @media only screen and (max-width: 480px) { .features { flex-direction: column; } nav { flex-direction: column; } } /* Tablet styles */ @media only screen and (min-width: 481px) and (max-width: 768px) { .feature { flex-basis: calc(50% - 20px); } } /* Desktop styles */ @media only screen and (min-width: 769px) { .feature { flex-basis: calc(33.333% - 20px); } }

Responsive Images

Make images adapt to different screen sizes using CSS and the picture element.

Basic Responsive Images with CSS

img { max-width: 100%; height: auto; }

This makes images scale down if their container is smaller than the image's natural size.

Advanced Image Handling with <picture> Element

<picture> <source media="(min-width: 960px)" srcset="large-image.jpg"> <source media="(min-width: 480px)" srcset="medium-image.jpg"> <img src="small-image.jpg" alt="Responsive image"> </picture>

The picture element allows you to define different image sources for different viewport sizes.
