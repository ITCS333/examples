CSS Positioning Elements

Static Positioning (Default)

Elements with position: static follow the normal document flow. This is the default positioning for all HTML elements.

This is in the normal flow.
Static
This is also in the normal flow.
.static-box { position: static; /* The element follows normal document flow */ }

Relative Positioning

Elements with position: relative are positioned relative to their normal position in the document flow.

This is in the normal flow.
Relative
This is also in the normal flow.
.relative-box { position: relative; top: 20px; left: 20px; /* Offset from its normal position */ }

Absolute Positioning

Elements with position: absolute are removed from the normal document flow and positioned relative to their nearest positioned ancestor.

This is in the normal flow.
Absolute
The absolute box doesn't affect this element's position.
.absolute-box { position: absolute; top: 40px; right: 40px; /* Positioned relative to nearest positioned ancestor */ }

Fixed Positioning

Elements with position: fixed are removed from the normal document flow and positioned relative to the viewport. They stay in the same position even when the page is scrolled.

Fixed
.fixed-box { position: fixed; bottom: 20px; right: 20px; /* Always stays in the same position regardless of scroll */ }

Sticky Positioning

Elements with position: sticky are positioned based on the user's scroll position. They act like relative until a specified scroll position is met, then act like fixed.

.sticky-box { position: sticky; top: 0; /* Acts like relative positioning until the scroll position is reached */ /* Then acts like fixed positioning */ }
