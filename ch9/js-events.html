JavaScript Events Tutorial

Introduction to Events

JavaScript events are actions or occurrences that happen in the browser, such as a button click, page load, or key press. Events allow you to create interactive web pages by executing code when specific actions occur.

Key Concepts:
  • Event: An action that occurs in the browser
  • Event Handler: A function that executes when an event occurs
  • Event Listener: A method that attaches an event handler to an element
  • Event Object: Contains information about the event that occurred
Basic Event Handling
Example 1: Click Event

The most common event is the click event. Here's how to handle a button click:

// Method 1: Using addEventListener document.getElementById('myButton').addEventListener('click', function() { alert('Button clicked!'); }); // Method 2: Using onclick property document.getElementById('myButton').onclick = function() { alert('Button clicked!'); };

Mouse Events

JavaScript provides various mouse events to detect different mouse interactions:

  • click: When the element is clicked
  • dblclick: When the element is double-clicked
  • mousedown: When a mouse button is pressed down on the element
  • mouseup: When a mouse button is released over the element
  • mouseover: When the mouse pointer moves onto the element
  • mouseout: When the mouse pointer moves out of the element
  • mousemove: When the mouse pointer moves while it's over the element

Keyboard Events

The Event Object

When an event occurs, JavaScript automatically passes an event object to the event handler function. This object contains useful information about the event.

Example: Using the Event Object
// The event object is automatically passed to the event handler element.addEventListener('click', function(event) { // Common event object properties console.log(event.type); // Type of event (e.g., "click") console.log(event.target); // Element that triggered the event console.log(event.currentTarget); // Element that the event listener is attached to console.log(event.clientX); // X coordinate of the mouse pointer console.log(event.clientY); // Y coordinate of the mouse pointer // Prevent default behavior event.preventDefault(); // Stop event propagation event.stopPropagation(); });

Form Events

Form events are triggered when users interact with form elements:

  • submit: When a form is submitted
  • reset: When a form is reset
  • change: When the value of an input element changes
  • focus: When an element receives focus
  • blur: When an element loses focus
  • input: When the value of an input/textarea changes
Example: Form Events

const form = document.getElementById('demoForm'); const nameInput = document.getElementById('nameInput'); const commentInput = document.getElementById('commentInput'); const formLog = document.getElementById('formEventLog'); function logFormEvent(event) { formLog.innerHTML = `${event.type} event on ${event.target.name || event.target.id}

Event Propagation

When an event occurs on an element, it first runs the handlers on it, then on its parent, and so on up the DOM tree. This is called "bubbling".

const outer = document.getElementById('outer'); const middle = document.getElementById('middle'); const inner = document.getElementById('inner'); const propLog = document.getElementById('propagationLog'); function logPropagation(event) { propLog.innerHTML = `Event on ${event.currentTarget.id} (${event.eventPhase})
` + propLog.innerHTML; } // Bubbling phase (default) outer.addEventListener('click', logPropagation); middle.addEventListener('click', logPropagation); inner.addEventListener('click', logPropagation); // To stop propagation inner.addEventListener('click', function(event) { propLog.innerHTML = `Stopping propagation at INNER
` + propLog.innerHTML; event.stopPropagation(); }, false);
Event Phases:
  1. Capturing Phase: The event goes down to the element (rarely used)
  2. Target Phase: The event reaches the target element
  3. Bubbling Phase: The event bubbles up from the element

To use the capturing phase, set the third parameter of addEventListener to true.
